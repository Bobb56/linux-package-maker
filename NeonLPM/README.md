# NeonLPM 
NeonLPM est une version modifiée de Neon, augmentée de quelques fonctions dans le but
de prendre en charge les fonctionnalités d'un installateur.

Fonctions ajoutées par rapport à Neon de base :

- copyPath(path1, path2)
- deletePath(path)
- makeDirectory(path)
- getHomePath(path)
- makeExecutable(path)
- getCurrentPath()
